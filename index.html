Workshop 3: Job Market Data Analysis

In this project, I used Large Language Models (LLMs) in Google Sheets to extract structured information from job postings. The extracted dataset was then analyzed using AI tools to explore relationships between:

  • Experience requirements
  • Programming languages
  • Salary ranges

This project demonstrates how LLMs can automate data extraction workflows and support deeper exploratory data analysis.
